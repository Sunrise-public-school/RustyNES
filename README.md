## üéÆ Using RustyNES

After you launch RustyNES, you'll need some NES games to play. You can find game ROMs online. Make sure you have the right to download these games legally.

### Loading a Game

1. On the main screen of RustyNES, click on the "File" menu.
2. Select "Open" and navigate to the folder where you saved your NES games.
3. Choose the game file (it will usually have a `.nes` extension) and click "Open."
4. Your game will start, and you can use your keyboard or a game controller to play.
